Nani's Voice Assistant is a browser-based chatbot interface with voice control and speech synthesis built using HTML, CSS, JavaScript, and Python (Flask). The assistant can receive both typed and spoken inputs and respond via text and voice using an intelligent backend like OpenAI.

---

## Features

- Voice input using Web Speech API (SpeechRecognition)
- Text-to-speech replies (SpeechSynthesis)
- OpenAI-powered chatbot backend
- Mobile responsive design
- Space-themed UI with animations
- Dual-font design for user input and bot replies
- Typing and response feedback
- Fly-by rocket and floating planet animations
